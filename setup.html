Setup & Configuration

  1. After download the Zip file from Code Canyon, unzip the file, open AITranslator.sln with Visual Studio.
  2. At visual studio, click on View, then click on "SQL Server Object Explorer"
  3. Expand a localdb, right click on Databases folder, click on "Add New Database"
  4. Enter a database name, click OK.
  5. Expand the created database, right click on the database and click Properties
  6. Copy the connection string value
  7. Open appsettings.json, at "DefaultConnection", paste the connection string value that you have copied just now.
  8. Follow this video to create your OpenAI API key. Once you have copied your API key, go to appsettings.json and paste the API key for "OpenAiApiKey"
  9. Follow this video to find your timezone ID. Here are the list of timezone that you can select from. Copy the timezone ID and replace the "TimeZone" value in appsettings.json
  10. Follow this video to find your SMTP username and password. You can use SendGrid, Brevo (formerly SendInBlue) or any other SMTP providers. Copy the SMTP username, password, port number, and host. Replace the SMTP values in appsettings.json.
  11. Done! You can now press CTRL+F5 to build and run the project! After successfully running the project, click on register, and the first registered account will automatically be assigned the admin role.
  12. If you failed to build the project, you may try the solution stated here: https://auratheme.notion.site/auratheme/Not-able-to-build-the-project-d2bee0ce63674dffafead92dad2c7af0.
